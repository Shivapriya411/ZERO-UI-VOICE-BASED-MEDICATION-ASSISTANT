# Zero-UI Voice Based Medication Assistant

## Abstract

The Zero-UI Voice Based Medication Assistant is a Flutter-based healthcare application designed to help elderly and low-literacy users identify medicines using a smartphone camera and receive voice guidance in regional languages. The application minimizes user interaction by eliminating complex menus and typing, making medicine identification more accessible and user-friendly.

## Features

* Medicine identification using camera scanning
* Voice guidance using Text-to-Speech (TTS)
* Multi-language support
* Zero-UI interaction design
* Elderly-friendly and accessible interface
* Healthcare assistance for rural and low-literacy users
## Technologies Used
* Flutter
* Dart
* Visual Studio Code
* Android SDK
* Text-to-Speech (TTS)
* JSON-based Medicine Database

## System Architecture

User → Camera Scan → Medicine Recognition → Medicine Database → Text-to-Speech → Voice Output

## Project Objectives

* Identify commonly used medicines using camera input
* Provide voice-based guidance in multiple languages
* Reduce confusion caused by similar-looking medicines
* Assist elderly and low-literacy users
* Improve medication accessibility and safety

## Future Enhancements

* AI-powered medicine recognition
* OCR-based medicine strip reading
* Medicine dosage reminders
* Emergency contact alerts
* Cloud-based medicine database
* Personalized medication history

## Domain

Healthcare Technology
